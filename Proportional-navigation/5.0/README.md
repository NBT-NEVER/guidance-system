# 实验 5.0：比例导引法允许攻击区计算

## 1. 实验内容

本实验按照课程实验五的要求，在目标由原点出发并沿 x 轴正方向做匀速直线运动的条件下，扫描导弹相对目标的不同发射半径与不同发射方位角，考察比例导引法在最大可用过载受限时的命中能力。
这里的“允许攻击区”是指：导弹在实际过载已经按 `10g` 饱和限制后，仍能够实现脱靶量小于 `0.1m` 的全部初始发射点集合。

## 3. 实验条件

- 目标初始位置：`(0, 0) m`
- 目标速度：`400 m/s`
- 目标航向：沿 `x` 轴正方向
- 导弹速度：`500 m/s`
- 导弹初始速度方向：始终对准目标初始位置
- 最大可用过载：`10 g`
- 导引律：比例导引法
- 导航比：`N = 4`
- 仿真步长：`0.01 s`
- 发射半径范围：`rm = 125, 250, ..., 2500 m`
- 发射方位角范围：`qm = 0°, 5°, ..., 355°`

## 4. 计算思路

### 4.1 初始发射点建模

导弹发射点采用极坐标扫描方式给出：

\[
x_M(0)=r_m\cos q_m,\qquad y_M(0)=r_m\sin q_m
\]

其中 `rm` 为导弹距原点的距离，`qm` 为导弹发射点相对基准线的方位角。

### 4.2 比例导引指令

比例导引法采用

\[
n_c=\frac{N V_c \dot{\lambda}}{g}
\]

其中

\[
\dot{\lambda}=\frac{x_r\dot{y}_r-y_r\dot{x}_r}{r^2},\qquad
V_c=-\frac{x_r\dot{x}_r+y_r\dot{y}_r}{r}
\]

程序先计算理论需用过载 `n_c`，再按 `±10g` 做过载饱和，得到实际执行过载，并据此更新导弹航向和位置。

### 4.3 脱靶量判定

本实验的允许攻击区判据不是“理论需用过载是否小于 10g”，而是：

\[
R_{\min}<0.1 \text{ m}
\]

也就是说，只要在实际过载已经限幅的前提下，整条弹道的最小弹目距离小于 `0.1m`，就判定该发射点属于允许攻击区。

为避免 `0.01s` 步长下只看离散采样点最小距离而漏判步间命中，程序在每个时间步内额外估计相对位移线段上的连续最小距离，用它作为脱靶量统计依据。

## 5. 程序流程

1. 从 `config.py` 读取目标参数、导弹参数、扫描网格和输出路径。
2. 在 `main()` 中双重循环扫描每一个 `(rm, qm)` 初始发射点。
3. 对每一个发射点建立目标与导弹二维运动模型，并设置导弹初始速度方向对准目标。
4. 按比例导引律计算视线角速度、闭合速度、理论需用过载和实际限幅过载。
5. 统计最大需用过载、最大实际过载、飞行时间和最小脱靶量。
6. 输出攻击区结果表，并绘制直角坐标攻击区图与参数平面攻击区图。

## 6. 输出文件

- `outputs/tables/summary.csv`
- `outputs/tables/summary.json`
- `outputs/figures/attack_zone.png`
- `outputs/figures/attack_zone_parameter_plane.png`

其中：

- `attack_zone_parameter_plane.png` 表示发射半径与发射方位角参数平面上的允许攻击区。

## 7. 运行方法

```bash
python main.py
```

## 8. 结果分析关注点

- 位于目标运动前方的发射点，通常具有更长的调整时间，允许攻击区容易向目标前方延伸。
- 位于目标后方且初始视线角变化较快的发射点，更容易出现需用过载峰值较大的情况。
- 最大需用过载是评估导引负担的重要指标，但本实验的攻击区划分最终仍以“实际限幅后能否把脱靶量压到 `0.1m` 以下”为准。
