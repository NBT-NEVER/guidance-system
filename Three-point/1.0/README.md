# 实验一（中等）：三点法制导律参数对导引弹道特性的影响

## 1. 实验内容
本实验只研究三点法制导律参数变化对导引弹道特性的影响，不涉及前置量法、半前置量法、机动目标对比、攻击禁区和姿态回路。

- 导弹初始位置：`(1000, 3000) m`
- 目标初始位置：`(2500, 7500) m`
- 导弹初始速度：`(140, 420) m/s`
- 目标初始速度：`(240, 0) m/s`
- 目标运动形式：水平匀速直线运动
- 导弹最大可用过载：`3 g`
- 时间步长：`0.01 s`

本实验按单变量扫描方式分别改变三类参数：

- `k_e`：`0.1` 到 `3.0`
- `k_d`：`0.0` 到 `2.0`
- `k_c`：`0.0` 到 `2.0`

其余两项参数固定在基准值：

```text
k_e = 1.2
k_d = 0.8
k_c = 0.6
```

## 3. 计算模型

三点法这里采用地面站位于坐标原点的二维平面模型。目标视线角写成：

$$
\theta_t = \operatorname{atan2}(y_T, x_T)
$$

导弹相对参考线的线偏差写成：

$$
e = x_M \sin \theta_t - y_M \cos \theta_t
$$

再把线偏差、线偏差变化率和参考线角速度组合成导弹弹道角速度指令：

$$
\dot{\gamma}_c = -\left(k_e \frac{e}{R_t} + k_d \frac{\dot e}{V_M} + k_c \dot{\theta}_t\right)
$$

需用法向过载为：

$$
n_c = \frac{V_M \dot{\gamma}_c}{g}
$$

程序再把 `n_c` 限制在 `3 g` 范围内，得到实际参与积分的法向过载。

LaTeX 代码：

```latex
\theta_t = \operatorname{atan2}(y_T, x_T)
e = x_M \sin \theta_t - y_M \cos \theta_t
\dot{\gamma}_c = -\left(k_e \frac{e}{R_t} + k_d \frac{\dot e}{V_M} + k_c \dot{\theta}_t\right)
n_c = \frac{V_M \dot{\gamma}_c}{g}
```

## 4. 程序实现说明

本实验把真正的参数扫描和数值积分直接写在 `main()` 内，没有为其他实验保留接口函数。程序处理顺序是：

1. 先选定本轮扫描的是 `k_e`、`k_d` 还是 `k_c`
2. 当前扫描参数变化，其余两项保持基准值
3. 在每个时间步计算目标视线角、线偏差和线偏差变化率
4. 求导弹弹道角速度指令和需用过载
5. 对需用过载做 `3 g` 限幅
6. 更新导弹和目标位置
7. 记录最大需用过载、脱靶量和命中情况

主函数外只保留：

- 字体配置函数
- 结果保存函数
- 绘图函数

## 5. 运行方式

在本目录执行：

```bash
python main.py
```

## 6. 输出结果

程序运行后会生成：

- `outputs/tables/summary.csv`：三类增益扫描结果表
- `outputs/tables/summary.json`：结构化汇总结果
- `outputs/figures/metrics.png`：三类增益对应的最大需用过载和脱靶量曲线

## 7. 结果关注点

本实验主要看：

- 哪一类增益改变后更容易导致过载迅速增大
- 哪些参数范围内弹道更容易收敛
- 当过载受限为 `3 g` 时，不同参数对脱靶量的影响大小规律
