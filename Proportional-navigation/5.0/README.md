# 实验 5.0：比例导航法允许攻击区判定

## 1. 实验内容

本实验通过扫描不同发射距离和发射角，判定在固定导航比和固定速度比条件下，比例导航法的允许攻击区。这里的允许攻击区表示：导弹能够在给定过载约束下成功进入命中半径的初始发射区域。

## 2. 判定方法

比例导航指令为：

\[
n_c=\frac{K V_c \dot{\lambda}}{g}
\]

其中：

\[
\dot{\lambda}=\frac{x_r\dot{y}_r-y_r\dot{x}_r}{r^2},\qquad
V_c=-\frac{x_r\dot{x}_r+y_r\dot{y}_r}{r}
\]

对每个发射点，程序进行完整积分，并记录：

- 是否命中；
- 最大需用过载是否超过上限。

若满足命中且未超过过载限制，则判定该点为可行点。

## 3. 程序流程

1. 固定目标位置、目标速度、速度比和导航比。
2. 按发射半径与发射角构成网格。
3. 对每个网格点建立导弹初始位置。
4. 执行比例导航积分。
5. 输出攻击区散点图和汇总结果。

## 4. 输出文件

- `outputs/tables/summary.csv`
- `outputs/tables/summary.json`
- `outputs/figures/attack_zone.png`

## 5. 运行方法

```bash
python main.py
```

## 6. 结果观察重点

- 发射点离目标过近或过远时，都可能削弱有效拦截条件。
- 发射角不利时，初始视线角变化速度过大，会导致需用过载增大。
- 攻击区图能直接展示比例导航在给定参数下的可用拦截区域。
