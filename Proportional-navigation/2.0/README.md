# 实验 2.0：比例导航法中视线角速度测量噪声的影响

## 1. 实验内容

本实验研究比例导航法在存在视线角速度测量噪声时的拦截性能变化。比例导航制导律本质上直接使用视线角速度 \(\dot{\lambda}\) 生成法向加速度指令，因此测量噪声会被直接带入制导回路，进而影响轨迹平滑性、需用过载和脱靶量。

本版本是独立实验，只分析“噪声影响”，不包含其他比例导航实验模式。

## 2. 模型公式

### 2.1 相对运动关系

\[
\dot{\lambda}=\frac{x_r\dot{y}_r-y_r\dot{x}_r}{r^2}
\]

\[
V_c=-\frac{x_r\dot{x}_r+y_r\dot{y}_r}{r}
\]

其中：

- \(r\) 为弹目距离；
- \((x_r,y_r)\) 为目标相对导弹的位置分量；
- \((\dot{x}_r,\dot{y}_r)\) 为目标相对导弹的速度分量；
- \(V_c\) 为接近速度。

### 2.2 比例导航指令

\[
n_c=\frac{K V_c \dot{\lambda}}{g}
\]

其中 \(K\) 为导航比。

### 2.3 加入测量噪声

本实验不直接使用真实视线角速度，而使用带噪测量量：

\[
\dot{\lambda}_m=\dot{\lambda}+w
\]

其中 \(w\) 为零均值高斯白噪声，标准差由实验配置给出。于是制导指令变为：

\[
n_c=\frac{K V_c \dot{\lambda}_m}{g}
\]

## 3. 程序流程

1. 固定弹目初始几何关系与速度条件。
2. 选择多个导航比 \(K\)。
3. 对每个 \(K\) 再扫描不同噪声标准差。
4. 在 `main()` 中直接完成相对运动积分和噪声注入。
5. 记录最大需用过载、最大实际过载、飞行时间、脱靶量和过载饱和比例。

- `outputs/tables/summary.csv`

## 5. 运行方法

```bash
python main.py
```

## 6. 结果观察重点

- 噪声增大后，制导指令会更抖动，最大需用过载往往会上升。
- 在同样噪声水平下，不同导航比对噪声的放大程度不同。
- 当需用过载频繁触碰上限时，实际拦截效果会明显恶化。
