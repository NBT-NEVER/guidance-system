# 实验 6.0：带姿态回路的比例导航拦截对比

## 1. 实验内容

本实验比较比例导航法在“理想内环”和“带二阶姿态回路”两种情况下的拦截结果。外层制导律相同，但由于内环过载建立速度不同，导弹实际轨迹会发生明显变化。

## 2. 关键公式

### 2.1 外层比例导航

\[
n_c=\frac{K V_c \dot{\lambda}}{g}
\]

### 2.2 二阶姿态回路

\[
\ddot{n}+2\zeta\omega_n\dot{n}+\omega_n^2 n=\omega_n^2 n_c
\]

程序中写为：

\[
\ddot{n}=\omega_n^2(n_c-n)-2\zeta\omega_n\dot{n}
\]

再对 \(\dot n\) 与 \(n\) 逐步积分，从而得到实际过载。

## 3. 程序流程

1. 固定导航比、初始几何关系和速度参数。
2. 分别计算无姿态回路与带姿态回路两种工况。
3. 在 `main()` 内直接积分外层制导和内环动态。
4. 比较最大需用过载、最大实际过载、脱靶量和轨迹差异。

## 4. 输出文件

- `outputs/tables/summary.json`

## 5. 运行方法

```bash
python main.py
```

## 6. 结果观察重点

- 姿态回路引入滞后后，实际过载不能完全跟随外层指令。
- 在比例导航中，内环滞后会降低修正视线旋转的能力。
- 对比图可直接看出理想制导与实际控制回路之间的差异。
