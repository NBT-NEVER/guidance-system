# 实验一（容易）：比例导引法—导航比对导引弹道的影响

## 1. 实验内容
本实验只研究比例导引法中的导航比 `K` 对导引弹道的影响，不涉及噪声、目标机动、攻击区或姿态回路。

- 导弹初始位置：`(0, 0) m`
- 目标初始位置：`(10000, 3000) m`
- 导弹速度：`500 m/s`
- 目标速度：`300 m/s`
- 目标航向：沿 `x` 轴负方向匀速直线飞行
- 导弹最大可用过载：`20 g`
- 命中阈值：`0.1 m`
- 时间步长：`0.01 s`

扫描的导航比为：

```text
K = 1.5, 2.0, 2.5, 3.0, 3.5, 4.0, 5.0, 6.0
```

## 3. 计算模型

比例导引法先计算视线角速率：

$$
\dot{\lambda} = \frac{x_r \dot{y}_r - y_r \dot{x}_r}{r^2}
$$

其中：

- `r`：弹目距离
- `(x_r, y_r)`：目标相对导弹的位置分量
- `(\dot{x}_r, \dot{y}_r)`：目标相对导弹的速度分量

再计算接近速度：

$$
V_c = -\frac{x_r \dot{x}_r + y_r \dot{y}_r}{r}
$$

法向加速度指令写成：

$$
a_c = K V_c \dot{\lambda}
$$

程序中把 `a_c` 换算为过载并做 `20 g` 限幅，之后用于更新导弹弹道角。

LaTeX 代码：

```latex
\dot{\lambda} = \frac{x_r \dot{y}_r - y_r \dot{x}_r}{r^2}
V_c = -\frac{x_r \dot{x}_r + y_r \dot{y}_r}{r}
a_c = K V_c \dot{\lambda}
```

## 4. 程序实现说明

本实验把真正的比例导引计算直接写在 `main()` 内，不再额外封装成给其他实验复用的求解函数。主循环的处理顺序是：

1. 读取一个导航比 `K`
2. 初始化目标与导弹位置、速度
3. 逐步计算相对位置、相对速度、视线角速率和接近速度
4. 按比例导引律求需用过载并限幅
5. 更新导弹弹道角和弹目位置
6. 记录最大需用过载、最大实际过载、脱靶量和命中情况
7. 全部导航比计算完成后统一输出图表和结果表

主函数外只保留了：

- 字体配置函数
- 结果保存函数
- 绘图函数

## 5. 运行方式

在本目录执行：

```bash
python main.py
```

## 6. 输出结果

程序运行后会生成：

- `outputs/tables/summary.csv`：不同导航比对应的统计结果
- `outputs/tables/summary.json`：结构化汇总结果
- `outputs/figures/trajectory.png`：不同导航比下的弹道图
- `outputs/figures/metrics.png`：最大需用过载与脱靶量曲线

## 7. 结果关注点

本实验主要看：

- 导航比偏小时，弹道是否容易不收敛
- 导航比增大后，过载需求是否明显上升
- 哪个导航比区间下，比例导引既能命中又不过分增加过载
