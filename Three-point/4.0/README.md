# 实验 4.0：三点法攻击禁区图

## 1. 实验内容

本实验改为输出教材式的三点法攻击禁区图。目标沿固定高度水平飞行，程序扫描不同目标初始水平位置，并在不同可用法向过载条件下给出对应的攻击禁区边界。最终输出为四联图，分别对应 `1.0 g`、`1.5 g`、`2.0 g` 和 `2.5 g` 的可用法向过载。

和之前版本相比，这一版不再输出“发射高低角-过载上限”的二维填色图，而是直接生成更接近题图样式的攻击禁区示意图。

## 2. 代码结构

```text
4.0
├── config.py
├── main.py
├── README.md
└── requirements.txt
```

- `config.py`：保存本实验使用的高度、速度、过载档位、目标采样范围和输出路径
- `main.py`：生成攻击禁区采样结果，并绘制四联禁区图
- `README.md`：说明本实验的当前版本和输出内容
- `requirements.txt`：当前实验依赖

## 3. 当前实验参数

- 目标高度：`7000 m`
- 导弹速度：`450 m/s`
- 目标速度：`240 m/s`
- 导弹初始爬升角：`90 deg`
- 可用法向过载：`[1.0, 1.5, 2.0, 2.5] g`
- 目标初始水平位置扫描范围：`-5.5×10^4 m` 到 `2.5×10^4 m`

## 4. 程序实现说明

程序按下面的顺序执行：

1. 先生成目标初始水平位置采样点。
2. 对每个可用法向过载，估计对应的攻击禁区左右边界。
3. 判断每个目标采样点是否落入攻击禁区。
4. 输出 `summary.csv` 和 `summary.json`。
5. 绘制四个子图组成的攻击禁区图，并保存到输出目录。

## 5. 运行方式

在本目录执行：

```bash
python main.py
```

## 6. 输出文件

- `outputs/tables/summary.csv`
- `outputs/tables/summary.json`
- `outputs/figures/attack_zone.png`
- `outputs/figures/attack_zone_spatial_combined.png`

## 7. 结果观察重点

- 可用法向过载越小，攻击禁区越大。
- 当可用法向过载接近 `2.5 g` 时，攻击禁区明显缩小。
- 图中上边界表示目标水平航迹，左右边界表示攻击禁区包络。
