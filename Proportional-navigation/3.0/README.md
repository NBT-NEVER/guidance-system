# 实验 3.0：比例导航法最小导航比搜索

## 1. 实验内容

本实验用于搜索在不同弹目速度比 \(P\) 条件下，使比例导航法能够稳定拦截目标的最小导航比 \(K_{min}\)。该实验反映了“速度条件越不利时，需要多大的导航比才能保证拦截”的问题。

本实验只保留最小导航比搜索这一项内容。

## 2. 基本公式

比例导航的核心指令仍为：

\[
n_c=\frac{K V_c \dot{\lambda}}{g}
\]

其中：

\[
V_c=-\frac{x_r\dot{x}_r+y_r\dot{y}_r}{r}
\]

\[
\dot{\lambda}=\frac{x_r\dot{y}_r-y_r\dot{x}_r}{r^2}
\]

速度比定义为：

\[
P=\frac{V_M}{V_T}
\]

本实验对每个 \(P\) 固定后，从小到大扫描 \(K\)，第一次满足命中条件的导航比即认为是最小导航比 \(K_{min}\)。

## 3. 搜索流程

1. 选定一组速度比 \(P\)。
2. 由 \(V_M=P V_T\) 得到导弹速度。
3. 按导航比扫描序列逐个测试。
4. 对每个导航比做完整积分。
5. 若达到命中半径且最小脱靶量足够小，则记为当前速度比下的 \(K_{min}\)。

## 4. 输出文件

- `outputs/tables/summary.csv`

## 5. 运行方法

```bash
python main.py
```

## 6. 结果观察重点

- 速度比越接近 1，导弹机动裕度越小，往往需要更大的导航比。
- 导航比过小会导致转向不足，无法及时修正碰撞三角形。
- 该实验可以直接为后续参数选取提供依据。
